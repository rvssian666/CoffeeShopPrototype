# ☕ CoffeeShop Project

## 📋 Descripción del Proyecto

**CoffeeShop Project** es una aplicación web moderna diseñada para gestionar las operaciones diarias de una cafetería. Construida con las últimas tecnologías del mercado, ofrece una experiencia de usuario intuitiva y responsiva para administrar inventario, pedidos e información de clientes.

### ✨ Características Principales

- 🎨 **Interfaz moderna y responsiva** - Diseño adaptable a todos los dispositivos
- 📦 **Gestión de Inventario** - Control completo del stock y productos
- 🛒 **Gestión de Pedidos** - Sistema de órdenes intuitivo y eficiente
- 👥 **Gestión de Clientes** - Base de datos centralizada de clientes
- 📊 **Análisis y Reportes** - Visualización de datos con gráficos interactivos
- 🎭 **Temas personalizables** - Soporte para modo claro/oscuro
- ⚡ **Rendimiento optimizado** - Carga rápida y navegación fluida

## 🚀 Guía de Instalación

### Requisitos Previos

- **Node.js** ≥ 18.x
- **npm** ≥ 9.x o **pnpm** ≥ 8.x
- **Git**

### Instalación Paso a Paso

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/rvssian666/CoffeeShop-Project.git
   cd CoffeeShop-Project
   npm install
   ```

# o si usas pnpm

pnpm install
npm run dev

# La aplicación se abrirá en http://localhost:5173

```
